# PID Piper – Privacy Policy

_Last updated: 2026-05-05_

PID Piper is a desktop application that analyzes drone Blackbox flight logs
locally on your computer. We respect your privacy and collect the minimum
amount of data necessary to operate and improve the app.

## 1. What we process locally
- The Blackbox log files you open. **They never leave your computer.**
- Decoded CSV files generated by `blackbox_decode`. **They are written to the
  same folder as the source log file and never uploaded.**

## 2. What we collect remotely (telemetry)
PID Piper uses two third-party services.

### Sentry (error tracking)
- Service: <https://sentry.io>
- Region: EU (Frankfurt)
- Purpose: Capture unhandled errors and crash reports so we can fix bugs.
- Data: Stack traces, OS / Electron version, anonymous device id. We do
  **not** send file paths, log file contents, or personal information.

### PostHog (product analytics)
- Service: <https://posthog.com>
- Region: EU
- Purpose: Aggregate, anonymous usage statistics (e.g. "how many users
  open a Step Response chart").
- Data: Anonymous installation id, OS / app version, named events. We
  explicitly disable autocapture, page-view capture and session recording.
- We do **not** collect IP addresses, file names, or log contents.

## 3. Decoder downloads
On first use, PID Piper downloads two open-source binaries
(`blackbox_decode`, `blackbox_decode_INAV`) from a public GitHub release.
GitHub may log the IP of the download request as part of normal HTTP server
operation; PID Piper itself does not record this.

## 4. Your rights
Because we never collect personally identifying information, we cannot
correlate telemetry events to a specific individual. If you wish to opt out
entirely, you can:
- Block outbound traffic to `sentry.io` and `posthog.com` in your firewall, or
- Use a hosts file / DNS / ad-blocker / firewall to block those endpoints.

## 5. Contact
For privacy questions, please open an issue on the public GitHub repository.
